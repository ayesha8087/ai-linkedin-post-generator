# 💼 AI LinkedIn Post Generator

An AI-powered LinkedIn Post Generator built using Python and Streamlit.

This application helps users generate professional LinkedIn posts using simple inputs like topic, tone, and target audience.

---

# 🚀 Features

✔ AI-style LinkedIn post generation

✔ Multiple tone selection

✔ Professional formatting

✔ Download generated posts

✔ Simple and clean UI

✔ Fast local execution

---

# 🏗️ Tech Stack

* Python 🐍
* Streamlit 🎈

---

# 🧠 How It Works

1. User enters a topic
2. User selects a tone
3. User enters target audience
4. AI-style post is generated instantly
5. User can download the generated post
